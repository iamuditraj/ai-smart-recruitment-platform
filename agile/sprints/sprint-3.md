# AI-Based Smart Recruitment Platform

An AI-powered end-to-end recruitment system that automates resume screening, skill assessment, candidate ranking, and mock interview analysis.

---

## 📌 Project Overview

The system uses Artificial Intelligence and Natural Language Processing (NLP) to:

- Parse resumes automatically
- Extract candidate skills and experience
- Match resumes with job descriptions
- Conduct skill assessments
- Analyze mock interviews
- Rank candidates using a weighted scoring algorithm

---

## 🎯 Phase 1 Scope

Phase 1 focuses on building the core intelligence modules:

- Resume Upload & Parsing
- Skill Extraction using NLP
- Resume–Job Description Matching
- Candidate Ranking System
- Mock Interview Analysis
- Firebase-based Data Storage
